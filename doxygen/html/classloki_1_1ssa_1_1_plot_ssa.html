Detailed Description

Produces SSA-specific diagnostic plots for the loki_ssa pipeline.

Four plot types are provided:

  • scree : eigenvalue spectrum (log scale) + cumulative variance curve.
  • wcorr : r x r w-correlation matrix rendered as a colour heatmap.
  • components : first N elementary reconstructed components vs time (MJD).
  • reconstruction: original series + per-group reconstructions overlaid.

Output filenames follow the project convention: ssa_[dataset]_[parameter]_[plottype].[format]

All plots are gated by PlotConfig flags (ssaScree, ssaWCorr, ssaComponents, ssaReconstruction). Failures in individual plots are caught, logged as warnings, and do not abort remaining plots.

Scree plot: eigenvalue spectrum on log scale + cumulative variance.

Left y-axis: eigenvalues (log scale, bar chart). Right y-axis: cumulative variance fraction (line, 0-100%). A horizontal dashed line marks the 95% cumulative variance threshold.

Output: ssa_[dataset]_[param]_scree.[fmt]
