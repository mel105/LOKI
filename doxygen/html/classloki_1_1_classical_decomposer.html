Detailed Description

Classical additive decomposition: moving-average trend + per-slot seasonal.

Algorithm:

  1. Trend T[t]: centered moving average of width = period. NaN edges are filled by bfill/ffill via GapFiller (LINEAR).
  2. Detrended D[t] = Y[t] - T[t].
  3. Seasonal S[t]: for each slot s = (i mod period), compute the median (or mean) of all D[t] that fall in slot s, then normalize so that the sum over one period equals zero.
  4. Residual R[t] = Y[t] - T[t] - S[t].

The input series must be free of NaN. Use GapFiller in the pipeline before calling decompose().

Reference: Makridakis, Wheelwright & Hyndman (1998). Forecasting: Methods and Applications. Chapter 3.
