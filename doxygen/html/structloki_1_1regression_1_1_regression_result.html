Detailed Description

Result of a regression fit.

Populated by any Regressor::fit() call. All fields are always filled. The x-axis convention is: x = mjd - tRef, where tRef = mjd of the first valid observation. This ensures numerical stability and ms-resolution compatibility across all data types.

Prediction intervals require tRef and cofactorX – both are preserved here so that predict() can be called after fit() without re-fitting.
