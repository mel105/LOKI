Detailed Description

Holds all outputs produced by the Kalman filter pipeline.

Populated by KalmanAnalyzer::run(). Fields that are not computed (e.g. smoothed state when the RTS smoother is disabled, or forecast when steps == 0) are left as empty vectors.

All time vectors use MJD. State vectors contain only the first component of the state vector (position / level / velocity), which is the physically observable quantity in all three built-in models.
