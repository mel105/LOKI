Detailed Description

Generalised Extreme Value (GEV) distribution utilities.

Provides CDF, quantile, log-likelihood, MLE fitting, and return level computation for the three-parameter GEV used in Block Maxima analysis.

Parameterisation (von Mises form): F(x) = exp(-(1 + xi*(x-mu)/sigma)^(-1/xi)) for xi != 0 F(x) = exp(-exp(-(x-mu)/sigma)) for xi == 0 (Gumbel limit)

Special cases: xi > 0 : Frechet distribution (heavy tail) xi = 0 : Gumbel distribution xi < 0 : Weibull distribution (bounded upper tail)

Fitting uses MLE via Nelder-Mead.
