Detailed Description

Holds the three additive components of a decomposed time series.

The decomposition identity holds for every valid index i: original[i] == trend[i] + seasonal[i] + residual[i]

All three vectors have the same length as the input series passed to ClassicalDecomposer::decompose() or StlDecomposer::decompose(). No NaN values are present in any component – the decomposer guarantees this by requiring a fully gap-filled input series.
