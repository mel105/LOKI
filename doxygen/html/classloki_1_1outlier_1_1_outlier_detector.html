Detailed Description

Abstract base class for all outlier detectors.

Implements the Template Method pattern: detect() drives the full detection pipeline while derived classes supply the robust location/scale estimates, the per-point score function, the threshold, and the method name.

All detectors operate on a vector of residuals (deseasonalized values). They do NOT modify the input series – replacement is performed by OutlierCleaner via GapFiller.

◆ detect()

OutlierResult loki::outlier::OutlierDetector::detect ( const std::vector< double > & residuals) const

Runs outlier detection on a residual series.

Computes robust location and scale, scores every point, and collects those whose absolute score exceeds the threshold into OutlierResult. When scale == 0 (all values identical), returns an empty result.

Parameters
residualsInput residual series. Must not contain NaN.
Returns
OutlierResult with one OutlierPoint per detected outlier.
Exceptions
SeriesTooShortExceptionif residuals.size() < minSeriesLength().
MissingValueExceptionif any residual value is NaN.
