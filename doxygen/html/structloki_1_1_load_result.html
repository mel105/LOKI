std::vector<std::string> loki::LoadResult::columnNames

Column names extracted from the file header or generated automatically.

Parsed from a "% Columns: ..." comment line if present. Falls back to "col_2", "col_3", ... when no header is found. Index matches series: columnNames[i] describes series[i].
