Detailed Description

Augmented Dickey-Fuller (ADF) unit root test.

Tests H0: the series has a unit root (is non-stationary / I(1)). H1: the series is stationary (I(0)).

The ADF regression fitted is one of:

  • "none": dy_t = gamma*y_{t-1} + sum beta_i*dy_{t-i} + e_t
  • "constant": dy_t = alpha + gamma*y_{t-1} + sum beta_i*dy_{t-i} + e_t
  • "trend": dy_t = alpha + delta*t + gamma*y_{t-1} + sum beta_i*dy_{t-i} + e_t

The test statistic is the t-ratio of gamma (the coefficient on y_{t-1}). Critical values are from MacKinnon (1994) via adfCriticalValue().

Lag selection:

  • "aic" : minimise AIC over lags 0..maxLags
  • "bic" : minimise BIC over lags 0..maxLags
  • "fixed" : use exactly maxLags lags (maxLags must be >= 0)

When maxLags == -1 (auto), the upper bound is floor(12*(n/100)^0.25), the rule of thumb from Schwert (1989).
