Detailed Description

Piecewise cubic spline interpolant over a set of knots.

Given n knots (x[0] < x[1] < ... < x[n-1]) and corresponding values y[0..n-1], constructs a C^2-continuous piecewise cubic polynomial S such that S(x[i]) = y[i] for all i.

On each interval [x[i], x[i+1]] the spline is represented in the standard polynomial form:

S_i(t) = a[i] + b[i]*t + c[i]*t^2 + d[i]*t^3, t = x - x[i]

Coefficients are computed once during construction by solving a tridiagonal linear system. Evaluation is O(log n) via binary search for the interval.

Extrapolation beyond [x[0], x[n-1]] uses the cubic polynomial of the nearest end interval (no clamping to endpoint values).

Primary use cases in LOKI:

  • GapFiller::Strategy::SPLINE (smooth gap filling in TimeSeries)
  • SplineFilter (smoothing spline – needs derivative())
  • loki_spline app (general curve fitting)

Construction: use the static factory CubicSpline::interpolate().

Exceptions
SeriesTooShortExceptionif fewer than 3 knots are provided.
DataExceptionif x values are not strictly increasing, or if x and y have different sizes.
AlgorithmExceptionif the tridiagonal system is singular (should not occur for strictly increasing x).

Evaluates the second derivative of the spline at a single point.

S''_i(t) = 2*c[i] + 6*d[i]*t, t = xq - x[i]

The second derivative is piecewise linear and continuous at knots (C^2 property). Zero at both endpoints for NATURAL boundary condition.
