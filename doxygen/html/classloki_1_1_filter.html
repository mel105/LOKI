Detailed Description

Abstract base class for all time series filters.

Each concrete filter implements apply() and name(). Configuration is handled by a nested Config struct in each subclass.

Usage example:

MovingAverageFilter f{MovingAverageFilter::Config{.window = 365}};
FilterResult result = f.apply(series);
