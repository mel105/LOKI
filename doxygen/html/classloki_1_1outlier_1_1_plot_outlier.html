Detailed Description

Generates gnuplot diagnostic plots for the outlier detection pipeline.

Naming convention: [program]_[dataset]_[parameter]_[plottype].[format]

Plot catalogue: plotOriginalWithOutliers – original series + outlier markers plotCleaned – cleaned series line plot plotComparison – original vs cleaned overlay plotResiduals – residuals + outlier markers plotSeasonalOverlay – original + seasonal model overlay plotResidualsWithBounds – residuals + detection bound lines plotOutlierOverlay – pre + post detection on one plot (homogeneity) plotLeverages – DEH leverage h_ii vs time + threshold line plotAll – calls all enabled plots based on PlotConfig flags

Plots hat matrix leverage values h_ii over time with threshold line.

Output: [program]_[dataset]_[param]_leverage.[fmt]

The leverage vector has length n - arOrder; indices are offset by arOrder relative to the original series. Outlier positions (h_ii > threshold) are marked as filled triangles. The threshold line is drawn as a dashed red line.

Parameters
seriesOriginal input series (used for MJD timestamps and naming).
leveragesLeverage values h_ii, length n - arOrder.
thresholdDetection threshold chi2Quantile(1-alpha, p) / n.
arOrderAR lag order p (used as index offset and in title).
nOutliersNumber of detected outliers (for title).
