Detailed Description

Result of fitting a theoretical variogram model to empirical data.

Parameters are estimated via weighted least squares using Nelder-Mead. Weights are proportional to the number of pairs per bin (standard WLS).

For the "power" model the "sill" parameter stores the scaling coefficient and "range" stores the power exponent – non-standard, see krigingVariogram.hpp.

FUTURE: anisotropy parameters (angle, ratio) for spatial / space-time Kriging.
