Detailed Description

RAII wrapper around a gnuplot pipe.

Opens a persistent gnuplot process via popen() on construction and closes it cleanly on destruction. All gnuplot commands are sent through send() or the equivalent operator().

Usage example:

loki::Gnuplot gp;
gp("set terminal png");
gp("set output 'out.png'");
gp("plot 'data.dat' u 1:2 w l");
