Detailed Description

Leave-one-out cross-validation diagnostics.

Computed via the O(n^2) LOO shortcut (Dubrule 1983): e_i = alpha_i / [K^{-1}]_{ii} where alpha = K^{-1} * z. No re-factorisation per left-out point.

Ideal values: meanSE ~= 0 (unbiased estimator) meanSSE ~= 1 (correctly calibrated variance)

meanSSE > 1: variance underestimated (CI bands too narrow). meanSSE < 1: variance overestimated (CI bands too wide).
