Detailed Description

Orchestrates the full quality control pipeline for a single TimeSeries.

QcAnalyzer runs five independently-enabled diagnostic sections:

  1. Temporal coverage – gaps, completeness, sampling uniformity.
  2. Descriptive stats – mean, std, IQR, skewness, kurtosis, Hurst, JB test.
  3. Outlier detection – IQR, MAD, Z-score on MA-detrended residuals.
  4. Sampling rate – median step, non-uniform step fraction.
  5. Seasonal consistency – MedianYearSeries feasibility (auto-disabled sub-hourly).

No data modification is performed. The original series is left unchanged.

Outputs written to disk:

  • Text protocol: OUTPUT/PROTOCOLS/qc_[dataset]_[component]_protocol.txt
  • Flags CSV: OUTPUT/CSV/qc_[dataset]_[component]_flags.csv

Plots are generated by PlotQc and called from run() if the relevant PlotConfig flags are set.

Runs the full QC pipeline on a single series.

Sections are run in order: sampling -> temporal -> stats -> outlier -> seasonal. Sampling is run first so that medianStepSeconds is available to the other sections.
