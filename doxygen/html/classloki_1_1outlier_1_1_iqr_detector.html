Detailed Description

Detects outliers using the Interquartile Range (IQR) method.

A point x_i is flagged as an outlier when: |x_i - median(x)| / IQR(x) > multiplier

The default multiplier of 1.5 corresponds to the classic Tukey fence. The input series must be deseasonalized residuals; the detector does not account for seasonal structure.

Location estimate : median Scale estimate : IQR = Q3 - Q1
