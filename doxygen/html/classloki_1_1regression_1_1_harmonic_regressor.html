Detailed Description

Harmonic regression: y = a0 + sum_{k=1}^{K} [a_k*sin(2*pi*t/T_k) + b_k*cos(2*pi*t/T_k)].

Fits a mean offset plus K sine/cosine pairs, one per period in RegressionConfig::period (a single period) repeated harmonicTerms times as sub-harmonics: T, T/2, T/3, ..., T/K.

The x-axis is t = mjd - tRef (days relative to the first observation). Design matrix is built via DesignMatrix::harmonic().

Useful for modelling annual, semi-annual, and higher-order seasonal signals in climatological or GNSS time series.
