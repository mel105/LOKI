Detailed Description

Full output of one spectral analysis run on a single TimeSeries.

The frequencies / power vectors form the complete periodogram or PSD. amplitudes holds the amplitude spectrum (physical units = sqrt(2 * PSD * df)) and phases holds the phase angle in radians at each frequency bin. Both are filled by FftAnalyzer; for Lomb-Scargle, amplitudes = sqrt(power) and phases is empty. peaks contains only the top-N statistically significant entries.
