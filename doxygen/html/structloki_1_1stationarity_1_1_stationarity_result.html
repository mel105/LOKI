Detailed Description

Aggregated result of the full stationarity analysis pipeline.

isStationary reflects the joint conclusion:

  • ADF rejected (unit root rejected) AND KPSS not rejected (stationarity not rejected) -> stationary
  • ADF not rejected AND KPSS rejected -> non-stationary
  • conflicting or all disabled -> conclusion contains explanation

recommendedDiff is 0, 1, or 2 – the suggested differencing order d for ARIMA(p,d,q).
