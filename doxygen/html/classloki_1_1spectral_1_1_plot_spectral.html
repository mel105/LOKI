Detailed Description

Generates all spectral analysis plots via gnuplot.

Two plot types are available, each gated by a PlotConfig flag:

  • PSD / periodogram (plots.spectralPsd): power spectral density or normalised Lomb-Scargle periodogram on a log-log scale with dominant peaks annotated.
  • Spectrogram (plots.spectralSpectrogram): 2-D time-frequency heatmap produced by STFT. Only generated when SpectralConfig::spectrogram.enabled is true. X-axis: calendar time (MJD); Y-axis: period in days (log scale); colour: power (pm3d palette).

All output files follow the project naming convention: spectral_<dataset>_<component>_<plottype>.<format>

Temporary data files use the .tmp_ prefix and are deleted after each gnuplot call.
