Detailed Description

Describes a single station entry for spatial Kriging.

Used only when KrigingConfig::mode = "spatial" or "space_time". Not used in temporal Kriging.

PLACEHOLDER – spatial Kriging is not yet implemented. Spatial coordinates are expected in the same unit system as the data (e.g. decimal degrees for GNSS networks, metres for local surveys).
