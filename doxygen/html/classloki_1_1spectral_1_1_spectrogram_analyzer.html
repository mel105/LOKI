Detailed Description

Computes a Short-Time Fourier Transform (STFT) spectrogram.

Slides a windowed FFT across the signal with a configurable overlap, producing a 2-D power matrix indexed by [time_window][frequency_bin].

Algorithm

  1. Build the window function (same types as FftAnalyzer).
  2. For each frame starting at hop * frameIdx: a. Extract the segment of length windowLength. b. Apply window and compute single-sided periodogram via FftAnalyzer internals. c. Store in result.power[frameIdx].
  3. Frequency axis in cpd; time axis in MJD of each frame centre.
  4. Optionally restrict the frequency axis to the focus period window.

The output SpectrogramResult can be passed directly to PlotSpectral.
